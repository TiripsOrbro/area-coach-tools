# Area Coach Tools — desktop tray

Windows tray shell for **Area Coach Tools** (forecast, Excel build-to, daily reports, subscriptions, logs).

## Setup

```bash
cd "Y:\Taco Bell Dashboard\Area Coach Tools"
npm run desktop:install
npm run desktop:start
```

Open the UI at `http://localhost:3100/admin/`.

This app is standalone. It does not require `live-dashboard-app`.
